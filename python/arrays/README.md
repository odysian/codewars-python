# Arrays

Array-based algorithm solutions focusing on manipulation, filtering, and pattern detection.

## Subcategories

### array-manipulation/
- **fold-an-array.py** - Array folding and transformation
- **plus-one-array.py** - Array arithmetic operations

### array-filtering/
- **unique-in-order.py** - Array deduplication while maintaining order
- **moving-zeroes-to-end.py** - Array reordering with element filtering

### array-patterns/
- **lottery-ticket.py** - Pattern detection in arrays
- **length-of-missing-array.py** - Identifying patterns in array sequences

## Core Concepts

- **Array traversal** - Iterating through elements efficiently
- **In-place operations** - Modifying arrays without extra space
- **Pattern recognition** - Identifying regularities in array data
- **Element filtering** - Selective element removal or rearrangement

## Common Techniques

- Two-pointer approach for in-place operations
- Hash maps for frequency counting
- Sliding window for subarray problems
- Stack operations for nested structure processing
